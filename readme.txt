== Installation ==

1. Install the plugin like you always install plugins, either by uploading it via FTP or by using the "Add Plugin" function of WordPress. (Upload directory 'k_new_profil_area' to the '/wp-content/plugins/' directory)
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Activate the plugin at the plugin administration page
4. Settings for plugins edit or save WP user area (under users menu)
